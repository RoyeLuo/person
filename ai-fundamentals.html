📊 机器学习基础

📈 回归算法

  • K-Means - 快速分群
  • DBSCAN - 异常检测

⚠️ 异常检测

  • Isolation Forest - 工业常用
  • One-Class SVM - 单类分类
  • LOF - 局部异常因子
  • AutoEncoder - 深度学习方法

PyTorch 快速上手

import torch import torch.nn as nn from torch.utils.data import DataLoader # 数据准备 train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True) # 定义模型 class SimpleNN(nn.Module): def __init__(self): super().__init__() self.layers = nn.Sequential( nn.Linear(64, 128), nn.ReLU(), nn.Linear(128, 1) ) def forward(self, x): return self.layers(x) # 训练 model = SimpleNN() optimizer = torch.optim.Adam(model.parameters(), lr=0.001) criterion = nn.MSELoss() for epoch in range(100): for batch_x, batch_y in train_loader: optimizer.zero_grad() outputs = model(batch_x) loss = criterion(outputs, batch_y) loss.backward() optimizer.step()

🏭 工业AI实战要点

与互联网AI的区别：
  • 数据量小：工厂数据有限，需要小样本学习方法
  • 实时性高：需要边缘推理能力，毫秒级响应
  • 可解释性：决策需可解释，工程师需要理解原因
  • 容错率低：误报漏报代价大，需要人工复核机制

工业AI项目流程

  • 需求分析 - 明确业务痛点和ROI
  • 数据采集 - OPC UA/Modbus采集设备数据
  • 数据清洗 - 处理缺失值、异常值、噪声
  • 特征工程 - 提取有意义的工业特征
  • 模型训练 - 选择算法、调参、验证
  • 边缘部署 - ONNX导出、TensorRT优化
  • 运维监控 - 模型漂移检测、在线学习
