# 罗小添 - 个人项目展示站

## 🚀 项目简介

这是一个专为上位机工程师罗小添设计的个人项目展示网站，专注于展示工业自动化、半导体设备控制和MES系统开发经验。

## 📋 网站结构

```
xiaotianluo.github.io/
├── index.html          # 首页 - 个人简介与核心技能概览
├── tech.html           # 核心技术栈 - 详细技术能力介绍
├── projects.html       # 项目经验 - 完整的项目案例展示
├── career.html         # 职业经历 - 工作经历与教育背景
├── about.html          # 关于我 - 个人能力与自我评价
├── contact.html        # 联系我 - 联系方式与在线表单
├── css/
│   └── style.css       # 全局样式文件
├── js/
│   └── main.js         # 交互脚本
└── README.md           # 项目说明
```
