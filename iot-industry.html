🔌 工业通信协议

工厂设备互联的核心协议：

现场总线/工业以太网

  • OPC UA - 工业4.0标准
  • Profinet - 西门子
  • EtherNet/IP - 罗克韦尔
  • Modbus TCP/RTU - 通用
  • CC-Link - 三菱

物联网协议

  • MQTT - 轻量发布订阅
  • AMQP - 企业消息
  • CoAP - 资源受限设备
  • LWM2M - 设备管理
  • HTTP/REST - Web接口

数据格式

  • JSON - Web应用
  • Protobuf - 高效序列化
  • OPC UA - 信息模型
  • XML - 传统系统
  • 二进制 - 高性能
OPC UA 是工业4.0的核心：
  • 平台无关，支持Windows/Linux/嵌入式
  • 内置安全机制 (TLS + 用户认证)
  • 信息模型支持复杂数据结构
  • 发现服务，自动识别网络设备
  • 推荐学习：OPC Foundation .NET SDK 或 Python opcua
